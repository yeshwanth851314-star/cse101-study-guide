# CSE101 — C Programming Study Guide

Interactive study guide for **CSE101: Computer Programming** following the official syllabus.

## 📚 Content
- **Unit I** — Basics of C (character set, data types, all operators) — 80 Viva Q&A
- **Unit II** — Control Structures & I/O (if/switch/loops/break/continue/goto) — 80 Viva Q&A  
- **Unit III** — Functions & Storage Classes (recursion, math library, auto/static/extern) — 80 Viva Q&A
- **Unit IV** — Arrays (1D/2D, linear/binary search, bubble sort) — 80 Viva Q&A
- **Unit V** — Pointers, Dynamic Memory & Strings (malloc/calloc/realloc, string functions) — 80 Viva Q&A
- **Unit VI** — Structures, Unions & C++ basics — 80 Viva Q&A
- **Lab Practicals** — P1–P6 complete programs

## ✨ Features
- Theory with tables & notes from Kamthane, Balagurusamy & Deitel textbooks
- Syntax-highlighted C code examples
- MCQ practice with instant feedback & explanations
- Flash Card viva preparation (480 Q&A total, shuffle mode)
- 🤖 AI Tutor powered by Claude

## 🚀 Run Locally
```bash
npm install
npm run dev
```
